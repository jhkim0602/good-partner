# Good Partner Index

## Quick Start
1) Read PROTOCOL.md
2) Register people in registry/people.yaml
3) Create a project folder under projects/
4) Create work packets under projects/<project>/work/
5) Log sessions under projects/<project>/sessions/
6) Use handoffs when ownership changes

## Structure
- PROTOCOL.md: rules and naming
- registry/: people, projects list
- projects/: all project workspaces
- templates/: single document template set
- scripts/: automation
- adapters/: tool-specific integration templates
- USAGE.md: workflow guide
- REFERENCES.md: adapter alignment notes
