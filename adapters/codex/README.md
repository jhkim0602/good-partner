# Codex Adapter

Install the skill to enforce Good Partner rules in Codex.

## Install
- Copy `skills/good-partner` into `.codex/skills/` in your repo (team scope), or `~/.codex/skills/` (user scope).
- Ensure the skill is discoverable by Codex.

## Notes
- The skill points to good-partner/PROTOCOL.md as the source of truth.
